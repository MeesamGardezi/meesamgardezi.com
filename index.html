About

Not your average dev.

I'm Meesam — a self-taught builder from Pakistan who treats software like art. At 20, I've already launched many products that real humans actually pay for, shipped apps to 2,500+ users, and built AI systems that clients overseas rely on daily. No bootcamp. No team. Just obsession.

My toolkit? Flutter for the frontend magic, Node.js for the heavy lifting, Firebase for the real-time sauce, and a healthy dose of VPS wrangling, DNS wizardry, and deploying things at 3am. I don't just write code — I build entire products, from the first pixel to the last server config.

When I'm not shipping features, I'm probably overthinking UI animations or debating whether tabs or spaces are the right choice. (It's spaces. Fight me.)
